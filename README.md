# Portfólio Web - Versão em Português

Este repositório contém o portfólio pessoal de Salatiel Muzi Martins, desenvolvido com HTML, Tailwind CSS via CDN e JavaScript vanilla.

## Sobre

Página de apresentação pessoal com design responsivo, tema claro/escuro, menu mobile e modal de detalhes dos projetos.


## Tecnologias

- HTML5
- Tailwind CSS (CDN)
- JavaScript vanilla
- Lucide Icons
- Google Fonts

## Estrutura do projeto

```
Portifólio Web/
├── css/
│   └── styles.css
├── js/
│   └── script.js
└── index.html
```

## Observações

- A configuração do Tailwind está diretamente em `index.html` para manter as cores personalizadas e o dark mode.
- O arquivo `js/script.js` contém as funcionalidades interativas: alternância de tema, menu mobile, animação de digitação, scroll reveal e modal de projetos.
- O arquivo `css/styles.css` contém as customizações visuais que complementam o Tailwind.
